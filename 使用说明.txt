easy-claude-deepseek ── 使用说明
===================================

Claude Code + DeepSeek 一键部署工具
Windows 桌面安装程序，自动配置 Claude Code 使用 DeepSeek API。


一、使用方式
───────────

方式 1：运行安装程序（推荐）
───────────────────────────────

从 dist/ 目录找到 easy-claude-deepseek Setup 1.0.0.exe，双击安装。

步骤：
1. 填写你的 DeepSeek API Key（去 https://platform.deepseek.com 申请）
2. 点击「一键自动安装」
3. 程序自动完成：
   - 检测/安装 Node.js
   - 配置 npm 淘宝镜像（中国大陆加速）
   - 安装 cc-switch 网络切换工具
   - 安装 Git for Windows
   - 安装 @anthropic-ai/claude-code
   - 通过 cc-switch DeepLink 导入 DeepSeek 配置
4. 在 cc-switch 弹窗中确认导入
5. 打开新终端，输入 claude 即可使用


方式 2：独立 PowerShell 脚本（无需 GUI）
────────────────────────────────────────────

  .\install.ps1 -ApiKey "sk-xxxxxxxx"

不传 -ApiKey 则只检测环境，不安装。


方式 3：开发模式
──────────────────

  cd claude-code-deployer
  npm start


二、安装流程（5 步）
────────────────────

  ① 检测环境    检测 Node.js / npm，缺失则自动下载安装
  ② 配置源      npm registry 切换至淘宝镜像
  ③ 装工具      安装 cc-switch + Git for Windows
  ④ 装 Claude   npm install -g @anthropic-ai/claude-code
  ⑤ 配 Key      通过 cc-switch DeepLink 协议导入 DeepSeek 配置


三、原理
────────

Claude Code 使用 @anthropic-ai/sdk 连接 API。
cc-switch 拦截请求并路由到 DeepSeek 的 Anthropic 兼容接口：

  https://api.deepseek.com/anthropic

模型映射：

  ANTHROPIC_BASE_URL             = https://api.deepseek.com/anthropic
  ANTHROPIC_MODEL                = deepseek-v4-pro
  ANTHROPIC_DEFAULT_HAIKU_MODEL  = deepseek-v4-flash
  ANTHROPIC_DEFAULT_SONNET_MODEL = deepseek-v4-pro
  ANTHROPIC_DEFAULT_OPUS_MODEL   = deepseek-v4-pro


四、项目结构
────────────

  easy-claude-deepseek/
  ├── main.js                    Electron 主进程
  ├── preload.js                 安全 IPC 桥接
  ├── package.json               项目 + 构建配置
  ├── install.ps1                核心安装脚本
  ├── CC-Switch-v3.15.0-Windows.msi  离线安装包
  ├── renderer/index.html        暗色主题 UI
  ├── scripts/build.bat          构建脚本
  ├── README.md                  Markdown 说明
  └── 使用说明.txt               本文件


五、构建
────────

  npm run build        编译为 NSIS 安装包（dist/ 目录下）
  npm run build:dir    编译为免安装目录


六、注意事项
────────────

  - API Key 需要在 https://platform.deepseek.com 注册获取
  - 安装程序需要管理员权限（安装 MSI 需要）
  - cc-switch 是社区开源工具：github.com/farion1231/cc-switch
  - 首次安装后需在 cc-switch 弹窗中确认导入
  - 如需重新配置，打开 cc-switch GUI 切换供应商


常见问题
────────

  Q: 安装后 claude 命令不可用？
  A: 关掉当前终端，新开一个 cmd 或 PowerShell 再试。

  Q: claude 显示登录界面而不是直接进入？
  A: 打开 cc-switch → 检查供应商是否切换为 DeepSeek。

  Q: 在中国大陆下载速度慢？
  A: 安装程序已自动配置 npm 淘宝镜像。
